How to Use Your Excel Data

To use your Excel files with this alumni directory:

  1. Export each Excel file (2019-2024) as CSV
  2. Convert the CSV data to JSON format (you can use online converters)
  3. Replace the sample data in the alumniData object with your actual data
  4. Make sure the column names match exactly: name, photo, linkedin, company, designation, income, contact

For the photo column, you can either use emoji codes like in the sample data, or use URLs to actual photos.
